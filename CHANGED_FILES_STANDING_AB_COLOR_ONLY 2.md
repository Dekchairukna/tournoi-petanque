# แก้หน้า Standing A/B แบบสีพื้นตารางเท่านั้น

ไฟล์ที่แก้:
- `templates/event_standings.html`
- `app.py`

รายละเอียด:
- ข้อความสถานะกลับเป็นข้อความปกติ ไม่ใช้ badge สีเด่น
- เมื่อยังไม่ได้เลือกระบบ A/B: แสดงแค่ `เข้ารอบ`
- เมื่อเลือกระบบ `A/B หลัง Swiss`: แสดง `เข้ารอบ กลุ่ม A` หรือ `เข้ารอบ กลุ่ม B`
- สีพื้นตารางในหน้า Standing:
  - กลุ่ม A = ฟ้า
  - กลุ่ม B = เขียว
  - ตกรอบ = แดง
- Excel จะแสดงกลุ่ม A/B เฉพาะตอนดาวน์โหลดหลังเลือกระบบ A/B เท่านั้น
