# ปรับเอกสารพิมพ์ตารางเพลย์ออฟ A/B

## แก้ไขแล้ว
- ไม่วางรหัสทางเดินทีม `[L-A1]`, `[W-B1]` นำหน้าชื่อทีมในตารางพิมพ์
- ย้ายข้อมูลที่มาของทีมไปไว้ใต้ตารางแต่ละสาย เพื่อให้ชื่อทีมอ่านง่าย
- หน้าเว็บคีย์คะแนนย้ายรหัสทางเดินทีมลงใต้ชื่อทีมเช่นกัน
- บังคับไม่ให้การ์ดตารางหนึ่งสายถูกตัดครึ่งข้ามหน้า
- โหมด `ตารางทั้งหมด` แบ่งหน้าเป็นชุดละไม่เกิน 8 ตารางสำหรับ Knockout/A-B และไม่เกิน 4 ตารางสำหรับ Double Knockout
- เมื่อเปลี่ยนรอบหรือเปลี่ยนจากกลุ่ม A เป็นกลุ่ม B ระบบขึ้นแผ่นใหม่เสมอ
- เพิ่มหัวเอกสารแต่ละหน้า: ชื่อรายการ ประเภท รอบ และกลุ่ม A/B
- ปรับเส้นกรอบตารางเป็นเส้นดำเรียบร้อย อ่านง่ายเวลา Save as PDF
